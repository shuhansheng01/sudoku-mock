Rules

  1. Fill the grid so every row has all digits used once.
  2. Every column must also contain each digit once.
  3. Each sub-grid (3×3 in 9×9; 2×3 in 6×6) contains all allowed digits exactly once.
  4. Given numbers are fixed.
